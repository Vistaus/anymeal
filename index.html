AnyMeal

Recipe Database Software

AnyMeal is a fast recipe database software developed using SQLite3 and Qt5. It can manage a cookbook with more than 250,000 recipes, thereby allowing to import, search, display, and print them. AnyMeal is available for GNU/Linux and Microsoft Windows.
